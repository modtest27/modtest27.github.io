    HINFO lookup

      Find all host information for a domain name with this online DNS HINFO checker. For example, try mozilla.org or msn.com to view their HINFO records.

      How to lookup HINFO records

      Jump to

      HINFO records can be configured by the owner of a domain name. They are used to store information in the DNS, which can be looked up by anyone. Each record type has a different purpose.

      HINFO records are configured in DNS. But how can you look up these HINFO records? That depends on the operating system you are using. Windows, Linux and Mac all have a different way of finding HINFO records. The sections below contain instructions for each operating system.

      How to lookup HINFO records on Windows

      To check the HINFO records for a certain domain name on Windows, follow these steps:

      1. Open a command prompt by navigating to Start → 'Type here to search' → 'cmd' → Open.
      2. Type nslookup -q=hinfo example.com and hit [enter] to get the HINFO records for example.com.
      3. The HINFO records are listed below the Non-authoritative answer heading.

      How to lookup HINFO records on Mac OS

      To check the HINFO records for a certain domain name on a Mac, follow these steps:

      1. Open a terminal by entering [command] + [space] → 'terminal.app' → [enter].
      2. Type dig example.com hinfo and hit [enter] to get the HINFO records for example.com.
      3. The HINFO records are listed below the ANSWER SECTION heading.

      How to lookup HINFO records on Linux

      To check the HINFO records for a certain domain name on Linux, follow these steps:

      1. Open a terminal by entering [Super] → 'terminal' → [enter].
      2. Type dig example.com hinfo and hit [enter] to get the HINFO records for example.com.
      3. The HINFO records are listed below the ANSWER SECTION heading.

      How to find HINFO records

      To check the HINFO records of a domain, follow these steps:

      1. Open the HINFO lookup tool.
      2. Enter the domain name and hit [enter].
      3. The tool will query the HINFO records and shown them at the top of the page.

      How to lookup other record types

      There are other types of DNS records then just HINFO. The most commonly used DNS record types are A, AAAA, CNAME, NS, TXT, MX and SOA.
